СТРУКТУРА ПРОЕКТА TELEGRAM MONITOR
=====================================

ОСНОВНЫЕ ФАЙЛЫ:
----------------
main.py              - Главный файл запуска приложения
gui.py               - Графический интерфейс (tkinter)
config.py            - Модуль конфигурации
auth.py               - Модуль авторизации в Telegram
monitor.py            - Модуль мониторинга событий
database.py           - Модуль работы с базой данных SQLite
logger.py             - Модуль логирования

ПАПКИ:
------
handlers/             - Обработчики событий (создается автоматически)
logs/                 - Папка для логов (создается автоматически)
media/                - Папка для медиа файлов (создается автоматически)
sessions/             - Папка для сессий Telegram (создается автоматически)

КОНФИГУРАЦИОННЫЕ ФАЙЛЫ:
-----------------------
requirements.txt      - Зависимости Python
config_example.json   - Пример конфигурации
.env                  - Переменные окружения (создается пользователем)

ДОКУМЕНТАЦИЯ:
-------------
README.md             - Полная документация проекта
QUICKSTART.md         - Быстрый старт
PROJECT_STRUCTURE.txt - Этот файл

БАЗА ДАННЫХ:
------------
telegram_monitor.db   - SQLite база данных (создается автоматически)

ЗАПУСК:
-------
python main.py
